# Alexandria Phases

This folder is the team-facing work board.

## Folder Purpose

- `general/` contains the big-picture phase roadmap and team workflow.
- `Homer/` contains backend/project-management tasks.
- `Leira/` contains public frontend tasks.
- `Ethan/` contains auth/admin frontend and light backend support tasks.
- `Shane/` contains database/Supabase tasks.

## Agent-Facing Plans

Detailed implementation plans for each phase live in `.agent/plans/`.

Use both together:

1. Read `phases/general/roadmap.md` for the team picture.
2. Read your role's `tasks.md` for ownership.
3. Read the matching `.agent/plans/phase-*.md` before coding.
4. Update status only after the work is verified.
